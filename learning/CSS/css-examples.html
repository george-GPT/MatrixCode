Examples of CSS
Basic CSS tag
body { background-color: #f7f7f7; }
this sets the background color of the entire webpage to a light gray (hex color #f7f7f7)
Selector
.className { color: blue; }
targets elements with the class .className and sets their text color to blue
CSS property/value
h1 { font-size: 2em; }
sets the font size of all <h1> headings to 2em, which is twice the size of the current font size context
Box model
.box { margin: 10px; padding: 20px; border: 1px solid #000; }
this rule gives elements with class .box a margin of 10 pixels on all sides, padding of 20 pixels inside the border, and a solid black border with a width of 1 pixel
Flexbox
.flex-container { display: flex; justify-content: space-between; }
this makes the .flex-container a flexible box layout, distributing space between its children so that they are evenly spaced across the parent's main axis
Grid
.grid-container { display: grid; grid-template-columns: repeat(3, 1fr); }
this turns the .grid-container into a grid layout with three columns of equal width
Pseudo-class
a:hover { color: green; }
this rule changes the text color of links to green when the mouse pointer hovers over them
Media queries
@media (max-width: 600px) { .responsive { font-size: 14px; } }
this applies a font size of 14 pixels to elements with the .responsive class when the viewport width is 600 pixels or less
Transition
.transition { transition: background-color 0.5s ease; }
this rule adds an animation effect that gradually changes the background color of the element over half a second when it is altered
Animation
@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }
this defines an animation named fadeIn that changes an element's opacity from fully transparent to fully opaque
Unit
.unit { margin: 5vh 10vw; }
this sets the margin of an element using viewport height (vh) and viewport width (vw) units, which are relative to the size of the screen
Positioning
.position { position: absolute; top: 50px; left: 100px; }
this positions an element absolutely at 50 pixels from the top and 100 pixels from the left of its nearest positioned ancestor
Transform
.rotate { transform: rotate(45deg); }
this applies a transformation that rotates an element 45 degrees clockwise from its original position
Typography
.typography { font-family: 'Open Sans', sans-serif; line-height: 1.6; }
this rule sets the typeface of the text to 'Open Sans' with a fallback to sans-serif font and a line height that is 1.6 times the size of the font
List
ul { list-style-type: square; }
this changes the list item marker to a square instead of the default disc in an unordered list
Animation
.animate { animation: fadeIn 2s; }
this applies the fadeIn animation to an element so it transitions from transparent to opaque over 2 seconds
CSS Comment
/* This is a comment in CSS */
this is a comment that explains or annotates CSS code. Comments are ignored by the browser and do not affect how the page is styled
