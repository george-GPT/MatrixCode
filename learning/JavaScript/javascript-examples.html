Examples of JavaScript
Variable
let userName = 'John';
userName is a variable that stores the value 'John'
Data type
let age = 30;
This is a number data type; age is a variable that stores the value 30
Operator
let isAdult = (age >= 18);
This is a comparison operator; isAdult is a variable that stores the value true
Conditional statement
if (isAdult) { console.log(userName + ' is an adult.'); }
Displays 'John is an adult.'
Loop
let toDoList = ['shop', 'email', 'clean'];
toDoList.forEach(function(task) { console.log(userName + ' needs to ' + task + '.'); });
Displays 'John needs to shop.', 'John needs to email.', and 'John needs to clean.'
Function
function displayTask(task) { console.log(userName + ' needs to ' + task + '.'); }
displayTask is a function that displays 'John needs to shop.', 'John needs to email.', and 'John needs to clean.'
Event handling
document.getElementById('myButton').addEventListener('click', function() { console.log(userName + ' clicked the button.'); });
Assuming there is a button element with id 'myButton' in the HTML; displays 'John clicked the button.'
DOM manipulation
document.getElementById('welcomeMessage').textContent = 'Welcome, ' + userName + '!';
Assuming there is a div element with id 'welcomeMessage' in the HTML; displays 'Welcome, John!'
Array
let tasksCompleted = 0;
toDoList.push('exercise');
Adds a new task; toDoList is an array that stores the values 'shop', 'email', 'clean', and 'exercise'
Object
let user = { name: 'John', age: 30, tasksCompleted: 0 };
user is an object that stores the properties name, age, and tasksCompleted
Class
class User {
constructor(name, age) {
this.name = name;
this.age = age;
this.tasksCompleted = 0;
}
}
let userJohn = new User('John', 30);
userJohn is an instance of the User class
ES6 feature
let greeting = `Hello, ${user.name}! You have ${user.tasksCompleted} tasks completed.`;
Displays 'Hello, John! You have 0 tasks completed.'
Error handling
try {
completeTask(user, 'email');
} catch (error) {
console.error('Error completing task: ' + error.message);
}
Displays 'Error completing task: completeTask is not a function'
Single-Line javaScript comment
// this is a single-line comment in JavaScript
Comments are used to explain code and are not executed
