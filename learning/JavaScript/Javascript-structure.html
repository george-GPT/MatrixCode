Beginner JS Structure


1. Dom References as Variables

Define references to HTML elements for easy access throughout the script.

const element = document.getElementById("elementId");
const elements = document.getElementsByClassName("className");
const elements = document.getElementsByTagName("tagName");
const element = document.querySelector("selector");
const elements = document.querySelectorAll("selector");


2. [Global Scope] Variable Declarations

Declare variables that need to be accessed globally across functions.

let globalVar = 42;


3. Function Definitions

Define the functions that encapsulate reusable logic.

function myFunction() { Your reusable logic here }


4. [Local Scope] Variable Declarations

Declare variables that are only needed locally within the function.

let localVar = 10;


5. DOM Manipulations

Modify or interact with the DOM elements.

element.innerText = "New text content"
element.innerHTML = "New HTML content"
parentElement.appendChild(newElement);
parentElement.removeChild(newElement);
element.setAttribute("src", "new-image.jpg");


6. Event Listeners

Attach event listeners to DOM elements for interaction.

addEventListener("click", function() { // event handling code here });


7. Initialize code such as event listeners

Perform any additional setup tasks, like setting up all event listeners in one place.

setUpEventListeners();


8. Include Comments and Documentation

Maintain code clarity, especially as projects grow in complexity.
