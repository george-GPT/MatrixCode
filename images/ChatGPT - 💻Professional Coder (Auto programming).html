This approach allows the image size to adjust based on the width of the viewport, making it more flexible across different devices.

Here's how you can adjust your CSS:

.image-container-small {
    max-width: 80vw; /* Adjust the maximum width to be relative to the viewport width */
    width: 100%; /* Ensure it takes up the full width of its container */
    margin: 0 auto; /* Center the image horizontally */
    text-align: center; /* Center the image within its container */
}

    max-width: 100%; /* Ensure the image stays within the container */
    height: auto; /* Maintain aspect ratio */

}
