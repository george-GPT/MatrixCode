HTML Basics


HTML, which stands for Hyper Text Markup Language. It is the standard markup language used to create web pages.

It's designed to describe the structure of a web page semantically, with a series of elements that instruct the browser on how to display the content. These elements act as labels for pieces of content, indicating whether something is a heading, a paragraph, a link, or another type of content.

HTML is a markup language. It's used to describe the content of a web page.





Cheat Sheet
Structural Tags Define webpage structure
< >, < / > opening tag, closing tag
<!DOCTYPE html> used at the beginning of html documents to set file type
<html> </html> creates an HTML document
<head> </head> sets title & other info that isn't displayed (meta tags placed here)
<body> </body> sets off the visible portion of the document
<title> </title> define webpage title
Semantic Tags Add meaning to content for better structure and accessibility
<header> </header> defines the header of a section or document
<nav> </nav> defines a set of navigation links
<main> </main> specifies the main content of a document
<section> </section> defines a section in a document
<article> </article> defines independent, self-contained content
<aside> </aside> defines content aside from the content it is placed in
<footer> </footer> defines the footer of a document or section
Meta Tags Defines Webpage structure - placed in <head>
<meta charset="UTF-8"> defines the character encoding for the HTML document
<meta name="viewport" content="width=device-width, initial-scale=1.0"> configures the viewport to scale for different devices
<meta name="description" content="Your description here"> provides a brief description of the page
<meta name="keywords" content="HTML, CSS, JavaScript"> specifies keywords for search engines
<meta name="author" content="Your Name"> defines the author of the document
Formatting Style text & elements
<paragraph> creates a new paragraph
<br> inserts a break line (carriage reeturn)
<blockquote> inputs content in a quote - indents text from both sides
<div> used to format block content with CSS
<span> used to format inline content with CSS
<p class="important">This is an important paragraph.</p> groups HTML elements together, to style them collectively
<div id="header">This is the header</div> provides a unique identifier for a specific HTML element
Text Tags Display text content
<pre> </pre> creates preformatted text
<h1> </h1> --> <h6> </h6> creates headlines -- H1=largest, H6=smallest
<tt> </tt> creates typewriter-style text
<code> </code> used to define source code, usually monospace
<cite> </cite> creates a citation, usually processed in italics
<address> </address> creates address section, usually processed in italics
<em> </em> emphasizes a word (usually processed in italics)
<strong> </strong> emphasizes a word (usually processed in bold)
<u> </u> underlined text
<s> </s> strikethrough text
<b> </b> creates bold text (should use <strong> instead)
<i> </i> creates italicized text (should use <em> instead)
<font size=?> </font> sets size of font - 1 to 7 (should use CSS instead)
<font color=?> </font> sets font color (should use CSS instead)
<font face=?> </font> defines the font used (should use CSS instead)
Links create hyperlink elements
<a href="URL">clickable text</a> creates a hyperlink to an external page
<a href="#EXAMPLE">clickable text</a> creates a hyperlink within the page
<a href="mailto:EMAIL_ADDRESS">clickable text</a> creates a hyperlink to an email address
<a name="EXAMPLE"> creates a target location within a document
<p class="important">This is an important paragraph.</p> groups HTML elements together, to style them collectively
<div id="header">This is the header</div> provides a unique identifier for a specific HTML element
Lists Lists in HTML are used to organize and structure content
<ul> </ul> creates a bullet point list
<ul style="list-style-type: circle;"> creates an unordered list with circle hollow circle points
<ol> </ol> creates an ordered list
<ol type="A"> </ol> creates an ordered list with uppercase letters
<ol type="I"> </ol> creates an ordered list with roman numerals
<ol start=?> </ol> creates an ordered list starting at a specified value
<li> </li> encompasses each list item
<dl> </dl> creates a definition list
<dt> precedes each definition term
<dd> precedes each definition
Graphical elements Display images & media
<body style="background-color:COLOUR;"> sets background to specific colour
<hr> inserts a horizontal rule
Tables Organize data in rows/columns
<table> </table> creates a table
<tr> </tr> sets off each row in a table
<td> </td> sets off each cell in a row
<th> </th> sets off the table header (a normal cell with bold, centered text)
Table Attributes Modify table behavior
<table border=?> sets the width of the border around table cells
<table cellspacing=?> sets amount of space between table cells
<table cellpadding=?> sets amount of space between a cell's border and its contents
<table width=?> sets width of the table in pixels or as a percentage
<tr align=?> sets alignment for cells within the row (left/center/right)
<td align=?> sets alignment for cells (left/center/right)
<tr valign=?> sets vertical alignment for cells within the row (top/middle/bottom)
<td valign=?> sets vertical alignment for cell (top/middle/bottom)
<td rowspan=?> sets number of rows a cell should span (default=1)
<td colspan=?> sets number of columns a cell should span
<td nowrap> prevents lines within a cell from being broken to fit
<caption> sets a caption for the table
Forms Forms in HTML are used to collect and submit user input
<form> </form> defines a form
<select multiple name=? size=?> </select> Create a multi-select dropdownl
<select name=?> </select> creates a pulldown menu
<option> sets off each menu item
<label> provides a label for items in forms to improve accessibility
<textarea name=? cols="x" rows="y"></textarea> creates a text box area. Columns set the width; rows set the height
<input type="checkbox" name=? value=?> creates a checkbox
<input type="checkbox" name=? value=? checked> creates a checkbox which is pre-checked
<input type="radio" name=? value=?> creates a radio button
<input type="radio" name=? value=? checked> creates a radio button which is pre-checked
<input type="text" name=? size=?> creates a one-line text area. Size sets length, in characters
Input Tag Attributes Provides additional information about HTML elements
<input type="email" name=?> sets a single-line textbox for email addresses
<input type="url" name=?> sets a single-line textbox for URLs
<input type="number" name=?> sets a single-line textbox for a number
<input type="range" name=?> sets a single-line text box for a range of numbers
<input type="date/month/week/time" name=?> sets a text box with a calendar showing the date/month/week/time
<input type="search" name=?> sets a single-line text box for searching
<input type="color" name=?> sets a single-line text box for picking a color
HTML Entities encode special characters reserved for HTML code
&lt; represents the less-than sign (<) in HTML
&gt; represents the greater-than sign (>) in HTML
&amp; represents the ampersand (&) in HTML
&quot; represents a double quotation mark ("") in HTML
&apos; represents an apostrophe (') in HTML
&nbsp; represents a non-breaking space in HTML
HTML Comments
<!-- Comment --> adds an HTML comment not visible on the web page
